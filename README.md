# プロジェクトセコイ
プロセカを作(りたか)った

![thumbnail](screenshot_title.png)

## Features
- タイトル画面
- 曲選択画面
  - 難易度選択
  - オフセット，スピードの調整
- ゲーム
  - タップ，フリック，ホールド(スライド)
  - 3段階の判定とミス
- リザルト画面
